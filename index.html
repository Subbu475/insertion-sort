What is Insertion Sort?

Insertion Sort is a simple sorting algorithm that builds the sorted array one element at a time by comparing and inserting into its correct position.

Step-by-Step Process:

  • Step 1: Start with the first element (consider it sorted).
  • Step 2: Pick the next element and compare it backward with the sorted portion.
  • Step 3: Insert it into its correct position.
  • Step 4: Repeat until the entire array is sorted.
